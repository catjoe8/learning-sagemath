# sagemath-graph-theory-notes
Personal SageMath notebooks exploring graph theory and tree structures
relevant to phylogenetics.

These notebooks explore basic graph theory concepts using SageMath.

The work includes generating simple graphs such as paths and cycles,
computing graph invariants like vertex degree and chromatic number,
and experimenting with random trees and their distance matrices.

The later notebooks focus on interpreting trees as phylogenetic
structures by identifying and labeling leaves and exploring
weighted distances between them.
